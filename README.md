# Crypto Tracker
- A cryptocurrency price track app for Android devices.
### Features

>- It uses `coinpaprika` API for data. It is reliable, sustainable.
>-  Simple and easy-to-use UI.
>- Compatible with Android 7.0 Nougat and newer.
>- Java, Retrofit
